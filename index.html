Request Playground

You can open the developer console in this page to send requests.

Request function syntax:

request(relative-url, request-type, params);

Returns a promise, containing the response.

relative-url: a relative url of the endpoint.

request-type: "GET" or "POST".

params: the data sent in the request.
If the request-type is "GET", "params" must be a string in url encoding form. For "POST", the parameter is an object.

Examples

// POST Request
request("post","POST",{test1:0,test2:1}).then(response=>{
    console.log(response);
});

// GET Request
request("get","GET","test1=0&test2=1").then(response=>{
    console.log(response);
});
